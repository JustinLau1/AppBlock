#Android 应用程序屏蔽  

------

## 介绍

这是一个应用屏蔽程序，可以选择你想屏蔽的任何应用，让你放下手机，全心投入工作和学习。  

现在就开始屏蔽那些消耗你生命的应用，去完成了不起的计划吧。
