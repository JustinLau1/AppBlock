#Android 应用程序屏蔽


这是一个应用屏幕程序，可以选择你想屏蔽的任何应用，让你放下手机，全心投入工作和学习。  

现在就开始在工作时间屏蔽可能会消耗你生命的应用，去完成了不起的计划吧。
